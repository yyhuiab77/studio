# **App Name**: Lift Dashboard

## Core Features:

- Dashboard Command Center: Provides a real-time overview of active alerts, personnel status, and system uptime across all monitored assets.
- Incident Logging & Tracking: A comprehensive system for logging new incidents, tracking their status (e.g., pending, in progress, resolved), and managing resolution details.
- AI-Powered Incident Analysis: Utilizes Genkit as a tool to provide AI-driven summaries and probable cause analysis for complex incidents, aiding rapid diagnosis.
- Asset Inventory Management: A central registry for all vertical transport infrastructure assets, with dynamic status tracking based on incident and maintenance data.
- User Management: Secure control over user roles and permissions (e.g., 'Operations Manager', 'Technician') for different operational team members, including login functionality.
- Scheduled Maintenance View: Allows viewing of planned maintenance windows and schedules associated with specific assets, ensuring proactive management.

## Style Guidelines:

- Color Palette: A sophisticated and professional dark theme, meticulously crafted to convey precision and control. A refined Light Cerulean Blue (#68A7F7) serves as the primary accent for critical elements, calls to action, and highlights, symbolizing technological advancement and importance. The expansive background is rendered in a Deep Midnight Blue-Gray (#1C2025), offering a visually calm, low-glare canvas ideal for extended command center use. Strategic accents and alerts leverage a vibrant, yet elegant Violet (#6A26F7) to ensure immediate, stylish distinction without visual clutter.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, techy, and precise character, fitting for titles and dashboard metrics. Body font: 'Inter' (sans-serif) for high readability and neutrality, suitable for incident descriptions, logs, and general UI text.
- Utilize clean, modern outline icons from ShadCN UI (Lucide Icons). These icons should be functional and self-explanatory, supporting quick understanding of actions and status.
- A responsive, modular, and clear grid-based layout for the Dashboard Command Center, prioritizing crucial real-time data at the top. Incident and asset management screens will feature intuitive forms and searchable tables with logical information hierarchy. Navigational elements will be persistent and easy to access.
- Subtle, non-distracting animations for data updates (e.g., fading in new alerts), state changes (e.g., form submissions, status changes), and navigational transitions to provide a smooth, professional user experience.
